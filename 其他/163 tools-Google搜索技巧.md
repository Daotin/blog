## 精确搜索
只会返回和关键词完全吻合的搜索结果。
```
"前端bug"
```

## "-"搜索
用减号减去相关内容即可。
```
“前端” -百度百科
```

## "OR"搜索
通过OR 搜索, 可以得到和两个关键词分别相关的结果, 而不仅仅是和两个关键词都同时相关的结果。
```
Google OR 百度
```
